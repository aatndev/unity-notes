[SerializeField]: makes private variable access in inspector

static: makes the variable global-like; they belong to the class not the obj
const: constant var that you cant change or update later
readonly: only changeable while declaration or using constructer

| Keyword    | Changeable?             | When assigned?             | Needs compile-time value? | Static? |
| ---------- | ----------------------- | -------------------------- | ------------------------- | ------- |
| `static`   | YES                     | anytime                    | no                        | yes     |
| `const`    | NO                      | only at declaration        | yes                       | yes     |
| `readonly` | NO (after construction) | declaration or constructor | no                        | no      |
